Routes
All products by pages: /toys?page=NUMER
All products: /toys?perPage=10000
Producs by category: /category=category name
Add a product: /toys(POST)
Edit a product: /toys(PUT)
Delete a product: /toys(DELETE)
Requierd parameters in a valid toy: name:String, img:String, info:String, category:String, price:Number
Requierd parameters in a valid if you want to sign in as a new user : email:String, pass:String, name:String
